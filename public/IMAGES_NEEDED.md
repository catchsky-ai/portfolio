## 📥 **How to Download:**

### **Option 1: Google Images**
1. Go to Google Images
2. Search for the term (e.g., "BigQuery logo transparent")
3. Tools → Size → Large
4. Tools → Type → PNG (for transparency)
5. Download image
6. Rename to match filename above (e.g., `bigquery.webp`)
7. Place in `Portfolio-Website/public/images/`

### **Option 2: Official Websites**
- **AWS logos**: https://aws.amazon.com/architecture/icons/
- **GCP logos**: https://cloud.google.com/icons
- **Python logo**: https://www.python.org/community/logos/
- **Apache logos**: https://www.apache.org/foundation/press/kit/

## 🎨 **Image Optimization Tips:**

1. **Format**: Use .webp for better compression (or .png)
2. **Size**: 
   - Tech logos: 512x512px (square)
   - Achievement images: 800x600px (landscape)
3. **Transparency**: Use PNG with transparent background for logos
4. **File size**: Keep under 100KB per image for fast loading

## 🚀 **Quick Start:**

1. Create the folder if it doesn't exist:
   ```bash

   ```

2. Download images from Google
